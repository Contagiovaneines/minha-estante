# Politica de Privacidade - Minha Estante

Ultima atualizacao: 2026-05-20

## 1. O que e o Minha Estante

O Minha Estante e um app de biblioteca digital pessoal. Ele permite organizar, ler e ouvir arquivos de livros, HQs e audiobooks armazenados no proprio dispositivo do usuario.

## 2. Dados armazenados localmente

O app pode salvar no proprio aparelho:

| Dado | Finalidade |
|---|---|
| Nome de perfil | Exibicao na tela de perfil |
| Foto de perfil opcional | Exibicao no avatar do perfil |
| Lista de itens da biblioteca | Organizacao dos arquivos escolhidos pelo usuario |
| Progresso de leitura e audio | Retomar de onde parou |
| Progresso de TTS | Retomar leitura em voz alta |
| Favoritos, marcadores e status | Organizacao pessoal da biblioteca |
| Pastas locais vinculadas | Sincronizacao de arquivos escolhidos pelo usuario |
| Backup JSON exportado | Copia local da biblioteca e do progresso |

Esses dados ficam salvos no armazenamento local do app no dispositivo. Eles nao sao enviados automaticamente para servidor proprio.

## 3. Dados que o app nao coleta no fluxo atual

- O app nao cria conta online.
- O app nao coleta senha.
- O app nao usa backend proprio.
- O app nao vende dados.
- O app nao exibe anuncios.
- O app nao usa Google Analytics, Firebase Analytics ou SDK proprio de rastreamento no fluxo atual.

## 4. Acesso a arquivos

O app acessa arquivos e pastas que o usuario escolhe explicitamente pelo seletor de arquivos do Android.

O app usa esses arquivos para:

- catalogar a biblioteca;
- mostrar capa/metadados quando disponiveis;
- abrir livros, documentos, HQs e audios;
- salvar progresso local.

Remover um item da estante remove o registro do app, mas nao apaga automaticamente o arquivo original do dispositivo.

## 5. Texto para voz, OCR e traducao

O recurso de texto para voz usa o mecanismo TTS disponivel no dispositivo Android.

Se o recurso de OCR/traducao online estiver habilitado e o usuario usar esse fluxo, o texto reconhecido pode ser enviado por HTTPS a um servico externo de traducao para processamento. Nao use esse recurso com conteudo sensivel ou confidencial.

## 6. Backup

Ao usar exportacao de biblioteca, o app gera um arquivo JSON com itens e progresso. Esse arquivo fica no dispositivo ou no local escolhido pelo usuario e nao e enviado automaticamente para terceiros.

## 7. Exclusao de dados

O usuario pode remover dados das seguintes formas:

- Remover um livro da estante dentro do app.
- Limpar cache/dados locais quando a opcao estiver disponivel no app.
- Apagar dados do app nas configuracoes do Android.
- Desinstalar o app, o que remove os dados locais do app.

## 8. Seguranca

Os dados locais sao armazenados usando Hive e arquivos locais do app. O app nao aplica criptografia adicional propria aos dados locais.

Para maior seguranca, recomenda-se proteger o dispositivo com senha, PIN ou biometria.

## 9. Criancas

O app nao foi projetado especificamente para criancas e nao possui conta online ou coleta intencional de dados pessoais de criancas.
